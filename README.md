OpenAI GPTを使ったWordPress投稿・固定ページ・カスタム投稿・メディアからのSEOキーワード自動生成プラグイン

> 🎯 **AIパワーでSEO最適化を自動化！**

## 主な機能

- **AI自動キーワード抽出**: OpenAI GPT-4.1シリーズを使用したSEOキーワード自動生成
- **多様なコンテンツ対応**: 投稿・固定ページ・カスタム投稿・メディアファイルに対応
- **モデル選択可能**: GPT-4.1、GPT-4.1 Mini、GPT-4.1 Nano から選択
- **日本語最適化**: 日本語コンテンツに特化したキーワード抽出
- **リアルタイム生成**: 編集画面から即座にキーワード生成
- **設定可能な出力数**: 抽出キーワード数を自由に設定可能
- **インテリジェント機能**: 失敗したモデルの自動除外・復活機能とフォールバックリトライ

## 🚀 クイックスタート

### インストール
1. プラグインファイルをWordPressの`/wp-content/plugins/`ディレクトリにアップロード
2. WordPressの管理画面でプラグインを有効化
3. **設定** → **Kashiwazaki SEO Auto Keywords** からOpenAI APIキーを設定

### 基本設定
1. OpenAI Platform（https://platform.openai.com/）でAPIキーを取得
2. プラグイン設定画面でAPIキーを入力
3. 使用するGPTモデルを選択（推奨: GPT-4.1 Nano - 最も経済的）
4. 抽出するキーワード数を設定

## 使い方

### 投稿・固定ページでの使用
1. 投稿・固定ページの編集画面を開く
2. 右サイドバーの「Kashiwazaki SEO Auto Keywords」メタボックスを確認
3. 「キーワード抽出」ボタンをクリック
4. AIが生成したキーワードが自動で表示される
5. 必要に応じてキーワードを編集・追加

### カスタム投稿・メディアでの使用
- カスタム投稿タイプでも同様の手順でキーワード抽出が可能
- メディアファイル（画像・動画等）にもAI分析によるキーワード生成対応

## 技術仕様

### システム要件
- **WordPress**: 5.0以上
- **PHP**: 7.4以上
- **OpenAI API**: 有効なAPIキーが必要
- **インターネット接続**: AI API利用のため

### 対応GPTモデル
- **GPT-4.1**: 最高品質（コスト高）
- **GPT-4.1 Mini**: バランス型（推奨）
- **GPT-4.1 Nano**: 経済的（低コスト）

### APIコスト目安
- GPT-4.1 Nano: 最も経済的、日常利用に最適
- GPT-4.1 Mini: 品質とコストのバランス
- GPT-4.1: 最高品質、重要コンテンツ向け
